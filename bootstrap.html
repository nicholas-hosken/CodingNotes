BS GET STARTED

The below is a cheat sheet for setting up the html page for using Bootstrap. However, it is good practice to make sure that the below libaries are the most recent.

<!DOCTYPE html>
<html lang="en">
  <head>
   <meta charset="utf-8">
    <!-- Makes site responsive on mobiles -->
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <!-- Latest compiled and minified CSS -->
    <link rel="stylesheet" href=" http://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">
  </head>
  <body>
   <!-- jQuery library -->
   <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
   <!-- Latest compiled JavaScript -->
   <script src="http://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/js/bootstrap.min.js"></script>
  </body>
</html>

BS Grid Basic

    The Bootstrap grid system has four classes:
  • xs (for phones)
  • sm (for tablets)
  • md (for desktops)
  • lg (for larger desktops)

Choosing a class will cause all bigger screens to scale up the coloumns and all smaller screens to stack the coloumns on top of each other. However they neet to be in the same "row" div.

The following is a basic structure of a Bootstrap grid:

<div class="row">
  <div class="col-*-*"></div>
</div>
<div class="row">
  <div class="col-*-*"></div>
  <div class="col-*-*"></div>
  <div class="col-*-*"></div>
</div>
<div class="row">
  ...
</div>

BS Typography

The "small" tag is used to make a lighter secondary heading in a "H*" tag.

<h3>This is a heading <small>with a small tag.</small></h3>


The "mark" tag is used to highlight words

Use the mark element to <mark>highlight</mark> text.


The "abbr" tag is used to label abbriviations in text
