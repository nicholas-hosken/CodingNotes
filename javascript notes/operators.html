The = operator

The + operator

This operator can be used to add numbers; however it can also "concatenate" (add) strings.

The += operator

The assignment operator += can concatenate strings.

Note: Adding a number and a string, returns a string.

The -= operator subtracts a value from a variable.

The *= operator multiplies a variable.

The /= operator divides a variable.

The %= operator assigns a remainder to a variable.
