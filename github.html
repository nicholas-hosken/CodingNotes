Github

Instructional Video

First you will want to make a new repository at github.com and copy the HTTPS clone URL on the right

git

You can get a full list of git commands and what they do by simply typing: git

For a list of subcommands type: git help -a

For concept guides type: git help -g

dir (List Directories)

Simply type dir to list all directories within your current directory

cd (Change Directory)

To change down into a directory type: cd [filename]. To change up into a directory simply type: cd..

To change drive letters type: "[driveletter]:" Example: D:

git clone (Clones current code from github.com to your PC)

To get a copy of the repository onto your pc, access the directory to want to put it in and type: git clone [HTTPS clone URL]

When using the Gitshell app, you can right click to paste

git status

This command compares file tree in directory with file tree on Github and returns differences. Whether complete missing files or just simple modifications on a file.

git add

git add [filename] tells the cmd that we want this file to be committed in the next git commit. git add -a will add everything in the directory.

git commit

This command tells the cmd that we want this file to be sent to the repository in the next git push. Typing -m "[message]" tells cmd that we want to add a message to the commit. This message will be added to readme.txt in the repository on github.com.

(example: git commit -m "This is a message")

If you commit without a message you will get a pop up on the cmd; this is for multiline messages. To exit this pop up you will need to press Esc and type: :wq

git push

This command will take all files which have been committed and sync them up with the repository on github.com. Other developers can now use git pull to gain access to these files

git pull

This command will essentially clone all files on the github.com repository file tree that you dont currently have in the directory on your PC.

git reset

Removes everything which is waiting to be either committed or pushed.
